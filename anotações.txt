4 Git/Github básico
4.1 git clone git@..... ( clona o repo )
4.2 git status ( mostra o que tem de mudanças )
4.3 git add . ( adiciona as mudanças na area de stage )
4.4 git commit -m 'comment' ( commita as mudancas localmente )
4.5 git push ( sobre as mudanças ( commits locais ) para o GH )
4.6 git pull ( traz todas as mudanças que está no GH )
4.6 git diff ( use q para sair )
